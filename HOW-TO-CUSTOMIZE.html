üì∑ How to Add Your Photo

Step-by-Step Instructions:

  1. Prepare your photo
    • Use a professional headshot photo
    • Recommended size: 400x400 pixels (square)
    • Format: JPG or PNG
    • Keep file size under 500KB
  2. Save photo to project

    Place your photo in: public/profile-photo.jpg

  3. Update the About component

    Open file: src/components/ui/About.tsx

    Find this section (around line 24):

    <div className="w-48 h-48 bg-gray-200 rounded-full...">
  <!-- Placeholder icon -->
  <svg className="w-24 h-24 text-gray-400"...>
</div>

    Replace with:

    <img
  src="/profile-photo.jpg"
  alt="Omkar Jore"
  className="w-48 h-48 rounded-full object-cover mb-6 md:mb-0 md:mr-8 flex-shrink-0 shadow-lg"
/>
  4. Deploy changes 
    git add public/profile-photo.jpg src/components/ui/About.tsx
git commit -m "Add profile photo"
git push

    Wait ~2 minutes for Vercel to auto-deploy!

üöÄ How to Deploy Changes
